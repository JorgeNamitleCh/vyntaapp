Jorge hace todos los commits él mismo. NUNCA ejecutar `git add`, `git commit`, ni `git push`.

**Why:** Lo dijo explícitamente dos veces — primero en la sesión anterior, luego de nuevo cuando intenté hacer `git add` en esta sesión.

**How to apply:** Cuando el usuario pide "el commit" o "el mensaje del commit", entregar únicamente el texto del commit message en un bloque de código para que él lo copie. Nada más.
